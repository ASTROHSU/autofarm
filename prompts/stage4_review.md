# Stage 4：語言審核

## 你的任務

對兩段式摘要進行最終審核，確保語言品質與風格一致性。

---

## 兩輪審核

### 第一輪：語言在地化
- 逐字掃描是否有中國術語，列出問題並給出修正建議
- 確認台灣用語使用正確

### 第二輪：語氣一致性
檢查以下項目，逐條標注「✓ 通過」或「✗ 需修正」：

- [ ] 無預告句（「先說一下」「值得注意的是」）
- [ ] 無評論性語氣（「這顯示出...」「值得深思」）
- [ ] 無制式開頭（「隨著...」「在當今社會」）
- [ ] 加粗使用正確（只有人名、組織名、術語）
- [ ] 字數在 350 字以內
- [ ] 無簡體字或中國術語

---

## 輸出格式

```
## 審核結果

### 第一輪：語言在地化
[有問題列出修正，無問題說「通過」]

### 第二輪：語氣一致性
- 無預告句：✓ / ✗
- 無評論性語氣：✓ / ✗
- 無制式開頭：✓ / ✗
- 加粗使用正確：✓ / ✗
- 字數在限制內：✓（XXX 字） / ✗
- 無中國術語：✓ / ✗

### 修正後版本
[如有需要修正，直接輸出修正後的完整兩段]
```
